Giới Thiệu JavaScript

JavaScript là một ngôn ngữ lập trình phổ biến được sử dụng để tạo các trang web tương tác.

Biến và Kiểu Dữ Liệu

Trong JavaScript, bạn có thể khai báo biến bằng var, let hoặc const.

let x = 10; // Biến x có giá trị 10

Hàm

Hàm giúp bạn tổ chức mã lệnh và tái sử dụng.

function sayHello() {
    console.log("Xin chào!");
}

Toán tử

Toán tử dùng để thực hiện các phép toán trên giá trị và biến.

let a = 5 + 3; // Tổng là 8

Câu lệnh điều kiện

Sử dụng if-else để kiểm tra điều kiện.

if (x > 0) {
    console.log("Số dương");
} else {
    console.log("Số không dương");
}

Vòng lặp

Vòng lặp giúp thực thi mã nhiều lần.

for (let i = 0; i < 5; i++) {
    console.log(i);
}
