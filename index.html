Các công nghệ JavaScript đã học:

  • Biến và kiểu dữ liệu (let, const, var)
  • Hàm (function, arrow function)
  • DOM Manipulation
  • Sự kiện (Event Listener)
  • ES6+ (Destructuring, Spread, Modules)
  • Async/Await và Fetch API
  • LocalStorage & SessionStorage
  • Web APIs (Canvas, Fetch, Geolocation)
  • React Basics

Ví dụ JavaScript:

// Tạo nút bấm và thêm vào trang
document.addEventListener("DOMContentLoaded", () => {
    const btn = document.createElement("button");
    btn.textContent = "Bấm vào tôi";
    btn.onclick = () => alert("Bạn đã bấm nút!");
    document.body.appendChild(btn);
});
